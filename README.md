# pyrpa
pyrpa is a Python script and Fortran library for model calculations based on tight-binding Hamiltonians.
It can calculate the following physical properties:

- Band structure and DOS
- Fermi surface (2D/3D)
- Spectral function
- Boltzmann and linear-response transport (conductivity, thermopower, Hall coefficient)
- Spin susceptibility χ_s (RPA)
- Pairing susceptibility φ (superconducting channel)
- Self-energy via FLEX approximation
- Superconducting gap function via linearized Eliashberg equation
- Cyclotron effective mass
- de Haas–van Alphen (dHvA) frequency vs. field angle
- Carrier density

# Requirements
- Fortran compiler (tested: gfortran 13.3, AMD Flang (ROCm/aocc), ifx)
- LAPACK, BLAS, and FFTW (tested: OpenBLAS, MKL, AOCL (BLIS and Flame))
- OpenMP (for parallelization; bundled with the above compilers)
- Make
- Python 3.10 or later (tested: 3.13)
- Python standard library
  - ctypes
- Python external packages
  - numpy
  - scipy
  - scikit-image
  - matplotlib

# How to compile
1. Clone this repository.
2. Enter the `libs` directory.
3. Run `make`, specifying the compiler and math library via variables if needed:

   ```bash
   make FC=<compiler> SL=<library>
   ```

   | Variable | Valid values | Default |
   |----------|-------------|---------|
   | `FC` | `gfortran`, `ifx`, `ifort`, `flang`, `amdflang` | `flang` |
   | `SL` | `MKL`, `AOCL`, `OpenBLAS` | `MKL` |

   Example:
   ```bash
   make FC=gfortran SL=OpenBLAS
   ```

4. `libfmod.so` is generated in the `libs` directory.

To clean build artifacts:
```bash
make clean
```
